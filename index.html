Biography: About me!

My name is Joshua Jacob, and I am currently a student at Georgia Institute of Technology.

I am studying Computer Science and plan to obtain a minor in Economics.

My interests include application development, algorithmic programming, information security, working out, and playing piano.

I am currently involved in the Residence Hall Association Justice Board as a Justice, GreyH@t Information Security Club, the Catholic Center, and GT EcoCAR 3

SFTP Client

Created a Secure File Transfer Protocol Client to transfer files between remote and local filesystems. Used Paramiko to handle the SSH connection, and created a user friendly user interface using Gtk. This was built as a plugin for the KingPhisher toolkit.

Compare Campaigns Tool

Created a tool packaged in the KingPhisher suite that generates graphs to compare multiple data points of phishing campaigns in a user friendly way. This allows the user to select and deselct which campaigns they would like to compare and dynamically generates the graphs in real time.

View details »

vPong

Created a vPython implementation of the classic Pong game. Implements realistic forces and interaction between surfaces and actively keeps score. 2 player control enabled and handled by a user input parsing engine that edits a dynamic environment with multiple objects.
